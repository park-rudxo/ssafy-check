# SSAFY 출석 체크 알리미 (크롬 익스텐션)

SSAFY 교육생이 **입실(09:00 이전)·퇴실(18:00 이후)** 체크를 놓치지 않도록 도와주는 크롬 익스텐션입니다.

> 퇴실 버튼은 미리 눌러도 되지만, **18:00 이후에 누른 기록이 있어야** 정상 퇴실로 인정됩니다. 18시 이전 기록만 있으면 조퇴 처리될 수 있습니다.

## 기능

### 🟥 빨간 사각형 강조 (콘텐츠 스크립트)
`edu.ssafy.com` 접속 시 시간대와 출석 상태를 판단해서 눌러야 할 버튼을 **깜빡이는 빨간 사각형**으로 강조하고, 화면 상단에 경고 배너(남은 시간 카운트다운 포함)를 띄웁니다.

| 상황 | 동작 |
|---|---|
| 평일 09:00 이전 + 미입실 | 입실 버튼 빨간 강조 + "입실 체크하세요" 배너 (남은 시간 표시) |
| 평일 09:00 이후 + 미입실 | 입실 버튼 빨간 강조 + 지각 경고 배너 |
| 평일 18:00 이후 + 오늘 18시 이후 퇴실 클릭 기록 없음 | 퇴실하기 버튼 빨간 강조 + "지금 퇴실 버튼을 누르세요" 배너 |
| 평일 09:00~18:00 (입실 완료) | 강조 없음, 조용히 대기 |

### 🛡️ 조퇴 방지 안내
- 퇴실 버튼 클릭 시각을 기록해 두고, **18:00 이후 클릭 기록이 생길 때까지** 퇴실 버튼을 계속 강조합니다. 18시 전에 미리 눌러둔 기록만으로는 강조가 꺼지지 않습니다.
- 18:00 이전에 퇴실 버튼을 누르면(막지는 않음) "18시 이후에 한 번 더 눌러야 인정된다"는 안내 토스트를 띄웁니다.

### 🔔 크롬 알림 (백그라운드)
SSAFY 페이지를 열어두지 않아도 평일에 알림이 옵니다. 알림을 클릭하면 SSAFY 출석 페이지가 열립니다.

- **08:50** — "09:00 전에 입실 체크하세요"
- **18:00** — "퇴실 버튼을 누르세요"

> 크롬이 실행 중이어야 알림이 옵니다. 주말(토·일)에는 강조/알림이 동작하지 않습니다.

## 설치 방법

1. 이 저장소를 다운로드(또는 `git clone`) 합니다.
2. 크롬 주소창에 `chrome://extensions` 입력 후 이동합니다.
3. 우측 상단 **개발자 모드**를 켭니다.
4. **압축해제된 확장 프로그램을 로드합니다** 버튼을 누르고, 이 저장소 폴더를 선택합니다.
5. `edu.ssafy.com`에 접속하면 자동으로 동작합니다.

## 파일 구성

```
manifest.json   # 익스텐션 설정 (Manifest V3)
content.js      # 출석 상태 감지 + 빨간 사각형 강조 + 조퇴 방지 가드
content.css     # 빨간 사각형/배너 스타일 (깜빡임 애니메이션)
background.js   # 08:50 / 18:00 크롬 알림 스케줄러
popup.html/js   # 툴바 아이콘 클릭 시 현재 상태 요약 팝업
icons/          # 익스텐션 아이콘
```

## 동작 원리

- 페이지의 텍스트("입실", "퇴실하기", "정상 출석")를 기반으로 버튼과 출석 상태를 찾기 때문에 SSAFY 페이지의 세부 구조가 조금 바뀌어도 동작합니다.
- 15초마다 + DOM 변경 시마다 상태를 다시 확인합니다.

## 주의사항

- 이 익스텐션은 **자동으로 버튼을 눌러주지 않습니다.** 출석 체크는 본인이 직접 해야 하며, 익스텐션은 놓치지 않도록 시각적으로 알려주는 보조 도구입니다.
- 공휴일은 인식하지 못합니다 (평일이면 항상 동작).
