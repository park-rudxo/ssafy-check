# 📌 SSAFY 출석 체크 알리미 — 설치하고 쓰는 법

입실(오전 9시 전)·퇴실(오후 6시 후)을 놓치지 않게, SSAFY 사이트의 **입실/퇴실 버튼 위에 빨간 네모 박스**를 띄워주는 크롬 확장 프로그램이에요. 크롬(또는 엣지)에서 5분이면 설치돼요.

## 1. 파일 내려받기

1. 링크 접속 → https://github.com/park-rudxo/ssafy-check
2. 초록색 **`< > Code`** 버튼 클릭 → **`Download ZIP`**
3. 받은 ZIP 압축 풀기 (폴더 안에 `manifest.json` 파일이 바로 보이면 정상이에요)

## 2. 크롬에 설치

1. 주소창에 `chrome://extensions` 입력 후 이동해요
2. 오른쪽 위 **개발자 모드** 토글을 켜요
3. 왼쪽 위 **압축해제된 확장 프로그램을 로드합니다** 클릭해요
4. 방금 압축 푼 **`ssafy-check` 폴더**를 선택해요 (`manifest.json`이 들어있는 폴더예요)
5. 목록에 빨간 아이콘 "SSAFY 출석 체크 알리미"가 뜨면 끝이에요!

## 3. 사용

- `edu.ssafy.com` **홈 화면**(https://edu.ssafy.com/edu/main/index.do)에 접속하면 자동으로 동작해요.
- **입실 안 했을 때** → 입실 버튼에 빨간 박스가 떠요 (9시까지 남은 시간 표시). 입실 누르면 사라져요.
- **낮 시간** → 퇴실 버튼에 주황 박스가 떠요 ("18시 이후에 누르세요").
- **18시 이후** → 퇴실 버튼에 빨간 박스가 떠요. 18시 넘어서 퇴실 누르면 사라져요.
- 크롬이 켜져 있으면 아침 8:50 / 저녁 6시에 알림도 와요. (첫 실행 때 알림 **허용**을 눌러주세요)

## 🔄 업데이트

- 새 버전이 나오면 크롬 알림으로 **"새 버전이 나왔어요"** 라고 자동으로 알려줘요. (확장이 GitHub의 최신 배포를 주기적으로 확인해요)
- 알림을 클릭하거나, 확장 팝업의 **`업데이트 확인`** 버튼을 누르면 다운로드 페이지가 열려요.
- 업데이트하는 법: 최신 ZIP을 다시 받아 압축을 풀고(같은 위치에 덮어쓰기), `chrome://extensions`에서 새로고침(🔄) 한 번 눌러주면 끝이에요.
- (처음 설치한 버전에는 이 알림 기능이 없을 수 있어요. 이번에 한 번만 최신으로 업데이트하면 이후부터는 알림을 받게 돼요.)

## ⚠️ 참고

- 자동으로 눌러주는 건 아니고, **놓치지 말라고 표시만** 해주는 거예요. 출석은 직접 눌러야 해요!
- 박스는 출석 위젯이 있는 **홈 화면에서만** 떠요. 커리큘럼·강의실 같은 다른 화면에는 표시되지 않아요.
- 설치 후 박스가 안 보이면: 확장 목록에서 새로고침(🔄) 한 번 누르고 → SSAFY 홈 화면도 새로고침(F5)해주세요.
- 크롬을 껐다 켜도 유지돼요. 삭제하고 싶으면 `chrome://extensions`에서 제거하면 돼요.
